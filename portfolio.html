PaperClip2

Application mobile - Jeu

Flutter Java SQLite Google Play

Le projet

PaperClip2 est un jeu d'optimisation inspiré par le thought experiment de Nick Bostrom sur l'alignement de l'intelligence artificielle. Le joueur doit maximiser la production de trombones tout en gérant un système complexe de ressources, recherche et automatisation.

Défis et solutions

Le principal défi était de créer un système de progression non-linéaire avec des interactions complexes entre différents mécanismes de jeu. Nous avons mis en place une architecture modulaire qui permet au jeu d'évoluer et de se complexifier au fil du temps, avec des systèmes interconnectés qui créent une expérience émergente.

Technologies utilisées

  • Framework Flutter pour une interface réactive et fluide
  • Architecture Clean pour une maintenance facilitée
  • SQLite pour la persistance des données
  • Système d'équilibrage dynamique pour une progression optimale

Résultats

PaperClip2 a connu un succès d'estime auprès des joueurs intéressés par les jeux de réflexion et d'optimisation, avec une note moyenne de 4.7/5 sur les stores et une communauté active de joueurs partageant leurs stratégies.

CryptoTracker Pro

Application mobile - Finance

Flutter Hive DB Provider API CoinGecko API Binance

Le projet

CryptoTracker Pro est une application complète de suivi de portefeuille de cryptomonnaies qui permet aux utilisateurs d'importer leurs transactions, de suivre l'évolution de leurs investissements en temps réel et d'analyser leurs performances.

Défis et solutions

Les principaux défis étaient la gestion de données complexes et la synchronisation avec plusieurs API externes, tout en garantissant une expérience fluide même hors connexion. Nous avons mis en place une architecture Clean avec un système de cache robuste utilisant Hive, permettant une utilisation optimale même sans internet.

Technologies utilisées

  • Flutter pour une interface utilisateur réactive et multi-plateforme
  • Architecture Clean pour une séparation claire des responsabilités
  • Provider pour la gestion d'état
  • Hive pour le stockage local et le cache
  • Intégration des API CoinGecko et Binance

Résultats

L'application a permis aux utilisateurs de gérer efficacement leurs investissements en cryptomonnaies, avec des fonctionnalités avancées comme l'import de transactions depuis CSV/Excel, le support d'API d'échanges et une interface intuitive. Les utilisateurs apprécient particulièrement la possibilité d'utiliser l'application hors ligne et la sécurité offerte par l'authentification biométrique.

Système de gestion de bibliothèque

Logiciel métier

Java JavaFX SQL Architecture MVC

Le projet

Développement d'un logiciel de gestion complet pour une bibliothèque municipale, permettant le suivi des emprunts, la gestion du catalogue, les inscriptions des usagers et l'analyse statistique de la fréquentation.

Défis et solutions

Le principal défi était de remplacer un système obsolète tout en migrant les données existantes et en formant le personnel à la nouvelle interface. Nous avons mis en place une approche progressive avec des sessions de formation personnalisées et un système d'aide contextuelle intégré.

Technologies utilisées

  • Java et JavaFX pour une interface desktop responsive
  • Base de données SQL pour le stockage des données
  • Architecture MVC pour une maintenance facilitée
  • Système de rapports et statistiques personnalisables

Résultats

Le nouveau système a permis d'automatiser de nombreuses tâches administratives, réduisant le temps consacré à la gestion de 40%. Les usagers bénéficient maintenant d'un système de réservation en ligne et de notifications automatiques. Le personnel apprécie particulièrement la simplicité d'utilisation et les fonctionnalités d'analyse qui leur permettent de mieux comprendre les habitudes de lecture de leurs usagers.

Boutique en ligne d'artisanat local

Site web - E-commerce

HTML/CSS JavaScript PHP Responsive Design

Le projet

Création d'une boutique en ligne pour un collectif d'artisans locaux, leur permettant de vendre leurs créations directement aux consommateurs sans passer par des intermédiaires.

Défis et solutions

Le principal défi était de créer une expérience utilisateur élégante qui mette en valeur le caractère artisanal des produits, tout en offrant une gestion simplifiée pour des artisans peu familiers avec les outils numériques. Nous avons développé une interface d'administration intuitive avec des tutoriels intégrés.

Technologies utilisées

  • HTML/CSS et JavaScript pour le frontend
  • PHP et MySQL pour le backend
  • Design responsive adapté à tous les appareils
  • Optimisation SEO pour une bonne visibilité

Résultats

Le site a permis aux artisans d'augmenter leur chiffre d'affaires de 35% dès la première année, en atteignant une clientèle plus large. La solution est facilement maintenue par les artisans eux-mêmes, qui peuvent ajouter leurs nouveaux produits sans assistance technique.
